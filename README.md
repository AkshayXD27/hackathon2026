# CodeZenith Hackathon 2026 Project

##  Overview

This project was built as part of a CodeZenith hackathon (DSCASC).

##  Features

* 🔐 User Authentication (Login / Signup)
* 📊 Dashboard with user-specific data
* 🤖 AI-powered features (API-based)
* ☁️ Firebase integration (Database + Auth)
* ⚡ Fast and responsive UI
* 🌐 Clean and simple navigation

---

## Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Firebase
* **Database:** Firestore / MONGO DB
* **Authentication:** Firebase Auth
* **AI Integration:** 4 API-based models

---

## Project Structure

```
hackathon2026/
│── api/            # contains backend files handeled by vercel
│── assests/               # images and video
│── firebase-config.js # Firebase setup
│── index.html       # Entry point
│── dashboard.html       # dashboard
│── dashboard.js       # handles dashboard funcs & connects to apis and db
│── style.css       # styling of  pages
│── script.js       # handles login / sign up & connects firebase
│── README.md
```
